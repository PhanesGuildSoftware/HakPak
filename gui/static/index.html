Tools

Help

  • Install/Uninstall: Use the buttons on each tool to install or remove it. Installed tools also get an Update button.
  • Update All: Use the top Update All to refresh all installed tools.
  • Methods: Default is auto. A Native badge means your package manager has that tool.
  • Run tools: Open a terminal and run the tool by name (e.g., nmap --help, gobuster --help, sqlmap --help). Many tools provide usage guides via --help or their official docs.
  • Popular docs: Nmap · sqlmap · Gobuster · ffuf · Wireshark · hashcat · Nikto
  • Wireshark capture: To capture without sudo: sudo usermod -aG wireshark $USER && newgrp wireshark.
  • Kali repo (apt): Use Repo controls below to add/status/remove optional Kali repo with safe pinning.
  • CLI equivalents: hakpak2 list, hakpak2 install <tools...>, hakpak2 update [tool|all], hakpak2 uninstall <tools...>.
  • State & paths: App lives in /opt/hakpak2; links in /usr/local/bin. See hakpak2 doctor.
